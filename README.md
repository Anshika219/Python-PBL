
# 🚦 Traffic Accident Severity Prediction

## 📌 Project Overview
This project focuses on predicting the severity of traffic accidents using Machine Learning techniques.  
The system analyzes various factors such as speed, weather conditions, road type, and vehicle category to classify accident severity.

---

## 🎯 Objective
To build a predictive model that can:
- Identify accident severity levels
- Help in understanding key risk factors
- Support road safety analysis and decision-making

---

## 🧠 Machine Learning Models Used
- Logistic Regression
- Support Vector Machine (SVM)
  - Linear Kernel
  - Polynomial Kernel
  - RBF Kernel
- K-Nearest Neighbors (KNN)
- Random Forest Classifier

---

## ⚙️ Technologies & Libraries
- Python 🐍
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

---

## 🔄 Project Workflow
1. **Data Collection**
2. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical data
   - Removing irrelevant features
3. **Feature Selection**
   - Selected key factors like speed, weather, road condition
4. **Model Training**
5. **Model Evaluation**
   - Accuracy
   - Precision
   - Recall
   - F1-score
6. **Visualization**
   - Graphs and plots for analysis

---

## 📊 Results & Insights
- Random Forest performed best in terms of accuracy
- Weather and speed were major contributing factors
- SVM worked well with proper kernel selection
