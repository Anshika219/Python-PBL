# Python-PBL
# 🚦 Traffic Accident Severity Prediction

## 📌 Overview

This project focuses on predicting the **severity of traffic accidents** using key factors such as speed, weather conditions, road conditions, and vehicle type.
The goal is to build a machine learning system that can help in **analyzing accident risks and improving road safety**.

---

## 🧠 Methodology

### 🔹 Data Preprocessing

* Cleaned dataset by removing missing and irrelevant records
* Handled inconsistencies in data

### 🔹 Feature Selection

* Selected important features such as:

  * Speed
  * Weather conditions
  * Road conditions
  * Vehicle type

---

## 🤖 Models Used

* Logistic Regression
* Support Vector Machine (SVM):

  * Linear Kernel → for simple decision boundaries
  * Polynomial Kernel → for non-linear relationships
  * RBF Kernel → for complex patterns

---

## 📊 Model Evaluation

* Confusion Matrix
* Recall Score (important for identifying severe accidents correctly)
